# Day 21 Task – Student Portfolio Website Using HTML5 Semantic Tags

## Objective

The objective of this task is to understand and implement HTML5 semantic elements by developing a professional Student Portfolio Website.

Students should focus on creating a properly structured webpage using semantic tags instead of relying only on generic `<div>` elements.

---

# Learning Outcomes

By completing this task, you will learn:

* HTML5 page structure
* Usage of semantic tags
* Professional webpage organization
* Portfolio website development
* Basic responsive webpage planning
* Content structuring for real-world projects

---

# Task Description

Develop a **Student Portfolio Website** containing your personal and academic information.

The website should be created using proper HTML5 semantic elements.

---

# Mandatory Semantic Tags

Your project must include the following semantic tags:

```html
<header>
<nav>
<section>
<article>
<aside>
<figure>
<footer>
<main>
```

Use each tag appropriately according to its purpose.

---

# Semantic Tag Usage Example

## Suggested Structure

```html
<body>

    <header>
        <nav></nav>
    </header>

    <main>

        <section>
            <article></article>
        </section>

        <section>
            <article></article>
        </section>

        <aside></aside>

    </main>

    <footer></footer>

</body>
```
