# Day 22 Task – Enhancing Student Portfolio Website Using CSS

## Objective

The objective of this task is to learn the fundamentals of CSS and apply styling techniques to transform the HTML portfolio website into a professional and visually appealing webpage.

Students should focus on improving the design, layout, spacing, colors, and responsiveness of the portfolio created on Day 21.

---

# Learning Outcomes

By completing this task, students will learn:

* CSS Selectors
* Colors and Backgrounds
* Typography and Fonts
* Box Model (Margin, Padding, Border)
* CSS Units
* Flexbox Basics
* Positioning Concepts
* Hover Effects
* Responsive Web Design Fundamentals

---

# Task Description

Enhance the **Student Portfolio Website** developed on Day 21 by applying CSS styling.

The website should look professional and organized instead of a plain HTML page.

---

# Sections to Style

The following sections should be properly designed:

✅ Header Section
✅ Navigation Bar
✅ Hero Section
✅ About Section
✅ Education Section
✅ Skills Section
✅ Projects Section
✅ Certifications Section
✅ Contact Section
✅ Footer Section

---

# CSS Concepts to Implement

Students should utilize the following CSS concepts:

### 1. Colors and Backgrounds

* Background colors
* Text colors
* Gradient backgrounds (Optional)

---

### 2. Typography

* Font Family
* Font Size
* Font Weight
* Text Alignment
* Line Spacing

---

### 3. Box Model

Implement proper:

* Margin
* Padding
* Border
* Border Radius

---

### 4. Layout Design

Use:

* Flexbox
* Display Properties
* Width and Height Properties

---

### 5. Navigation Styling

The navigation bar should include:

* Proper alignment
* Hover effects
* Active link styling

---

### 6. Cards Design

Create professional cards for:

* Skills
* Projects
* Certifications

Cards should include:

* Padding
* Border Radius
* Shadow Effects
* Hover Effects

---

### 7. Images Styling

Apply styling to profile images:

* Circular profile image
* Proper sizing
* Border or shadow effects

---

### 8. Buttons Styling

Add buttons such as:

* Contact Me
* Download Resume

Include:

* Hover effects
* Smooth transitions

---

### 9. Responsive Design (Optional)

Make the website adapt to smaller screens by using:

* Relative units
* Flexbox wrapping
* Media Queries (Optional)
